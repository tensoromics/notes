# tensoromics/notes

Companion Jupyter notebooks for the TensorOmics _Notes_ section at
[tensoromics.com/notes](https://tensoromics.com/notes).

Each post on the site that includes code has a runnable notebook here. The post
explains the concept; the notebook lets you actually run it.

## Setup

```bash
git clone https://github.com/tensoromics/notes.git
cd notes
pip install -r requirements.txt
export ANTHROPIC_API_KEY=sk-ant-...
jupyter lab
```

Get an API key at [console.anthropic.com](https://console.anthropic.com/).

## What you'll need

- Python 3.10+
- An Anthropic API key
- About a dollar's worth of API credit per notebook (most cost a few cents)
